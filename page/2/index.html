深圳的“冬天”

发表于 2017-02-26 | 分类于 闲杂

深圳的冬天，是说不定什么时候就让你感受一下。昨天还阳光暖暖，短袖，短裤的，变天就是厚衣加身。深圳的冷是一种毫不防备的袭击。前几天还好好的，马上就让你手无所措。

我们被春天来了的假象所蒙蔽，没有这个强劲的冷空气南下，说不定就平稳过渡到夏天。但是偏偏要打击一下，那些以为熬过了深圳冬天的人，虽然立春已过，冬天还是时不时的闪现出来，寻找存在感。

对北方的同学说春天来了。北方同学诧异了，春天，没听说过，刚刚下的雪都还没有化。不给人希望，就没有失望。

晚上睡觉时，居然有蚊子在我耳边嗡嗡嗡。仿佛在嘲笑我，瞧你窝在被窝里的熊样，看爷裸奔，还用翅膀扇着风呢。没过多久，嗡嗡嗡声就没有了，估计应该是挂了吧

顺丰上市了

发表于 2017-02-24 | 分类于 闲杂

今天被顺丰上市刷屏了。真心替顺丰上市高兴，顺丰是个好的公司。

离开顺丰也快一年了，回想起来，在顺丰工作的日子，也还算不错的。认识了很多同事，有的现在都保持着很好的关系。刚开始是在顺丰做外包的，那一段时间加班很厉害，经常下班都是在11点，打车到家都是12点左右。估计是能力还能凑合，还很能加班的样子。于是乎，就转入顺丰内部了。

快递里面顺丰的IT是做得最好的，最新的方向，内部都是毫不避讳的去尝试。想当初，微信公众号刚刚开始，腾讯很多接口都不是很稳定，顺丰就开始开发微信公众号了，腾讯还特意来了两个码农，做技术支持。后面，微信公众号的反映还不错。

激进的未必都是好的，嘿客店的发展不如人意，当然了，也有好的，那时顺丰快递柜刚刚起步，老是没有用起来，经常被快递员当做储物柜了。现在的丰巢快递柜，还是不错的。

发展总是有这样那样的问题，但，总体还是向好的方向发展。

mac版本的一些无限期破解方法

发表于 2017-02-10 | 分类于 Terminal

1，显示系统隐藏文件。
在终端（Terminal）中输入如下命令：defaults write ~/Library/Preferences/com.apple.finder AppleShowAllFiles -bool true
（true 改成 false 就可以不再显示隐藏文件）需要重启，然后在finder中就可以看到隐藏文件了。

2，打开~/Library/Preferences/ByHost下的GlobalPreferences.XXXX.plist文件，XXXX为一串 数字，删掉”com.zennaware.Cornerstone:2.0”这一行，然后保存。（如果安装了XCode，会自动打开plist文件，否则 就得自己找个plist文件工具了，比如PlistEdit）

3，就此，安装记录已经删除，然后请重复方法一的步骤。
