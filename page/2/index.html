小马过河

发表于 2017-04-10 | 分类于 闲杂

有一天，一匹小马，来到河边，准备过河。突然，跳出一只松鼠，松鼠大叫道：“不要过去，危险，昨天我的朋友掉到这条河里，再也没有见到过他了。这条河的水很深。”说完，松鼠伤心得离开了这里。小马犹豫不决，不知道，应不应该过去。正在犹豫的时候，过来了一头老水牛，小马跑过去，问道：“水牛伯伯，这条河的水深吗？”，水牛回答道：“不深啊，我每天都从这里过。”水牛说完慢慢的走开了。小马更加犹豫了，还是不知道该不该过这条河，于是，小马决定跑回家问问妈妈。

“妈妈妈妈，路上我遇到一条河，松鼠说水很深，水牛伯伯说水不深，我不知道怎么办，该不该过河。”小马带着疑惑的眼神看着妈妈。妈妈说：“遇到事情，你自己先要有个判断，不管别人怎么说，要自己尝试了才知道啊。”小马的妈妈鼓励小马去试一试。

于是，小马又跑到河边，尝试着走到河里，小马小心翼翼的过完河后，感叹道：“哎，河水不像松鼠说的那么深，也不像水牛伯伯说的那么浅。”

又涨一岁，生活还得继续

不知不觉的，又涨了一岁，对于过生日来说，我是不敏感的。反而是身边的人，强调生日的意义，搞得我也视乎重视了起来。我认为，这一天就当做普普通通的一天过，也挺好的，没有必要强行把一年中的一天，或直几天，加上特别的意义。

最后，祝我生日快乐。我视乎也没有认为今天会比其他时间重要，但是多年前的今天，我来到这个世界，从此，这一天对我意义就变得不同，生活也就开始了，而且还得继续下去。

四月，清明节假期

发表于 2017-04-03 | 分类于 闲杂

四月伴随着一个要上班的愚人节来了。令人欣慰的是，马上就是清明节的三天假期。于是乎，我想起了那句清明节的诗，”清明时节雨纷纷，路上行人欲断魂。借问酒家何处有，牧童遥指杏花村。“

想想过去的人，对家庭的重视，视乎远远超过现在。现在越来越不在乎这种家庭感，是科技的发展让我们彼此之间走得更近，还是越来越远。古人的家书抵万金，现在的信息发达，唾手可得。古人送别故人，总是感叹，自此一别，又不知何时才能相见，然而今天，相见谁，视频聊天分分钟的事，即使是亲自相见，交通发达的今天，也不是难事。我们现在很难体会到古人对故人的那种情感，很长一段时间，没有任何消息，唯有彼此的牵挂。

先成为一个自律休息的人

发表于 2017-03-25 | 分类于 闲杂

前一段时间，有点失眠，导致精神不好，生活和工作都头疼。没有想到，休息不好，对人的精神和身体的影响都很大。在后面的调整中，我逐渐意识到，做一个自律的人，开始可能是很痛苦的事，但慢慢的开始调整过来了，有点小感悟。

对于一个自律的人，应该要做到什么呢。我不知道，我只是在调整的过程中，坚持按时睡觉，按时早起，做事的时候，不要沉溺其中，定时起来休息，调整。是的，我指的自律，并不是要完成自己所规定的什么任务，而是自律的休息。

人的精力是有限的，自律的休息是精力的保证。有了精气神，才能做好其他的除休息之外的事。做事的时候，不要沉溺其中，玩游戏的时候，不要沉迷其中，好的休息才是第一位的。每个人的适应都不是一样的，我是这样的，没有好的休息，再好的状态也出不来。也许，其他人不一样，要一直打鸡血才可以把事情做下。

因人而异吧，对我而言，先成为一个自律休息的人，很重要。

劝君再添一碗饭，莫在世间多挣扎

发表于 2017-03-21 | 分类于 闲杂

今天早上突然想到一句话，也不知道是什么原因，于是就写下来了。

有时候，觉得自己作，说好听一点就是折腾。作什么事都不那么安于本分，老想干点什么大事，尤其在晚上，开始构想宏图伟业，一直想着怎么干是最好，又怎么做，又达到目的，然后宏图伟业成了。哈哈哈。然而，并没有。

也许年龄大的时候，越会做事保守，激进只适合年轻人。我不太认同这种观点，保守也好，激进也好，只是做事的一种方式而已，正确的时间，选择正确的做事方式，才是处事之道。

难的是对事物的判断，有人在期货市场说，要是有人能够一个月都连续做出正确的判断，那么，他将成为这个世界上最富有的人。正真到判断决策的时候，又很难做出，是需要保守，还是需要激进的方式。有人说性格决定命运，有些人的性格天生都是激进的，于是乎，他们经常大起大落，结局好的就偶尔落到高点，差的话就落到低谷。保守的性格，大多就会落到平庸，平平淡淡也没有什么不好，小富即安是很多人的追求。

人中龙凤毕竟是极少的，他们风起云涌，也有大起大落。就像，秋风扫过树林一样，一阵风带起无数片树叶，有的树叶在风中飘逸，有的树叶随风舞动，有的树叶就默默的看他们飞。风静了，所有的树叶都将落到地面。仿佛，什么都没有发生过。

人的一生是有限的，人的精力也是有限的，劝君再添一碗饭，莫在世间多挣扎。
