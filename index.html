香港回归二十周年

发表于 2017-07-01 | 分类于 闲杂

香港回归的那一年，我刚刚小学毕业（哦哦，暴露年龄了），为了迎接香港回归，本来定在7月的小升初考试，提前到了6月。也就是六年级的时间少了，然而小学到中学之间的假期就多了。

发小学毕业证的那天，小伙伴们都拿到了毕业证，唯独我没有拿到。我着急的赶到校长办公室询问， 校长说你交的照片少了一张，没有照片贴在毕业证上，让我回家去拿，然后我跑回家去拿，然而家里面并没有找到。于是，又跑回校长办公室，校长说，尺寸差不多的也行。于是，我又跑回家，然而家里面并没有。当我再次气喘吁吁的跑到校长办公室时，校长也无法，就把一张本来要存档的照片，贴到我的毕业证上。于是，我就可以拿着小学毕业证高喊，“我不是小学生了！”。

那一年，小平同志想亲眼见证香港回归的愿望没有实现。小平同志的骨灰伴随着鲜花，洒在了中华大地，一代伟人就这样离开我们。那一年，假期特别长，电视里面的动画片少的可怜，整天播放的是以怀恋小平同志为主题的刘邓大军扫荡全世界的电视剧。那一年，我趴在外公家的木窗边，看远山，听着电视里面播放小平同志的讲话，小平同志操着一口地道的四川口音说到：“不解放思想不行”。

六一儿童节

发表于 2017-06-01 | 分类于 闲杂

告别六一儿童节很多年，最后一个儿童节，那天是阴天还是大太阳，我忘记了。我咬着冰棒，拿过节的零花钱买的。那时是也小学的最后的一年。

年年儿童节，学校都组织活动，有学生表演的节目，也有老师表演的。那天，全校都到操场上集合，带着自己的板凳，排排坐。操场前有一个很宽的水泥楼梯，楼梯后有一平台，平台后面就进教学楼了。那个水泥楼梯和后面的平台就是我们的舞台。有轮到我们班表演吹竖笛的时候，就安排一两个实力选手到话筒边，其他就是和稀泥，反正我是和稀泥，从小没有音乐细胞。有一次，音乐老师被要求来一个节目，就跳了一个舞蹈，好像是“北京的金山上”什么的。

现在，我读的那所小学还在，已经装上了华丽的自动门，每次路过只能透过自动门看里面，还能看到那个很宽的水泥楼梯。

突然想看苏大爷的《赤壁赋》

发表于 2017-05-05 | 分类于 闲杂

不知道为什么，突然想到《赤壁赋》，想看看。摘抄到此。

壬戌之秋，七月既望，苏子与客泛舟游于赤壁之下。清风徐来，水波不兴。举酒属客，诵明月之诗，歌窈窕之章。少焉，月出于东山之上，徘徊于斗牛之间。白露横江，水光接天。纵一苇之所如，凌万顷之茫然。浩浩乎如冯虚御风，而不知其所止；飘飘乎如遗世独立，羽化而登仙。

于是饮酒乐甚，扣舷而歌之。歌曰：“桂棹兮兰桨，击空明兮溯流光。渺渺兮予怀，望美人兮天一方。”客有吹洞箫者，倚歌而和之。其声呜呜然，如怨如慕，如泣如诉，余音袅袅，不绝如缕。舞幽壑之潜蛟，泣孤舟之嫠妇。

苏子愀然，正襟危坐而问客曰：“何为其然也？”客曰：“‘月明星稀，乌鹊南飞’，此非曹孟德之诗乎？西望夏口，东望武昌，山川相缪，郁乎苍苍，此非孟德之困于周郎者乎？方其破荆州，下江陵，顺流而东也，舳舻千里，旌旗蔽空，酾酒临江，横槊赋诗，固一世之雄也，而今安在哉？况吾与子渔樵于江渚之上，侣鱼虾而友麋鹿，驾一叶之扁舟，举匏樽以相属。寄蜉蝣于天地，渺沧海之一粟。哀吾生之须臾，羡长江之无穷。挟飞仙以遨游，抱明月而长终。知不可乎骤得，托遗响于悲风。”

苏子曰：“客亦知夫水与月乎？逝者如斯，而未尝往也；盈虚者如彼，而卒莫消长也。盖将自其变者而观之，则天地曾不能以一瞬；自其不变者而观之，则物与我皆无尽也。而又何羡乎！且夫天地之间，物各有主，苟非吾之所有，虽一毫而莫取。惟江上之清风，与山间之明月，耳得之而为声，目遇之而成色，取之无禁，用之不竭，是造物者之无尽藏也，而吾与子之所共适。”

客喜而笑，洗盏更酌。肴核既尽，杯盘狼籍。相与枕藉乎舟中，不知东方之既白。
